# Show Respawn Zones

Allows you to visibly see respawn zones while in-game in [Nitro Turtles](https://store.steampowered.com/app/3952070/Nitro_Turtles/) Demo. Works on both maps, tested in game versions v0.3.3, although should work in any version.

# How to install
Refer to [this](https://github.com/NorthernChicken/Nitro-Turtles-Mod-Creator)
